Projetos

  • Projeto 1: Sistema de Gerenciamento de Tarefas

    Descrição: Este projeto é um sistema de gerenciamento de tarefas desenvolvido com Angular. Ele permite que os usuários criem, editem e excluam tarefas, além de marcar tarefas como concluídas. O sistema utiliza um backend em Node.js para armazenar os dados em um banco de dados MongoDB.

    Tecnologias Utilizadas: Angular, Node.js, Express, MongoDB

    Link para o Projeto: https://github.com/Rhavell

  • Projeto 2: Aplicativo de Receitas

    Descrição: Este projeto é um aplicativo de receitas onde os usuários podem procurar, visualizar e salvar suas receitas favoritas. O aplicativo possui uma interface amigável e responsiva, desenvolvida com Angular e Bootstrap. Os dados das receitas são obtidos de uma API externa.

    Tecnologias Utilizadas: Angular, Bootstrap, API de Receitas

    Link para o Projeto: https://github.com/Rhavell
